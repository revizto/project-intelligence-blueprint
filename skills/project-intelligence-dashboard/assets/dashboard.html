About

Everything on this dashboard is powered by the Revizto MCP Server — a secure, open interface that lets any compliant AI agent work with your project data in real time, in plain language.

Plain language
Ask your project anything

Natural-language answers across every role — owner to field engineer. No query language, no BIM expertise required.

Issues
Coordination & issue tracking

Query and filter issues by status, priority, assignee and type; see trends, comments and history; draft assignments — all approval-gated.

2D
Sheets & documents

Drawing-sheet metadata, version history and CDE sync status — so the right revision is always the one in hand.

3D
Clash & model coordination

Clash tests, groups and results across the federated model, ready for AI agents to reason over and route.

People
Members, roles & access

Team membership, project roles and licence detail — every tool call inherits the caller's permissions.

Insight
Live metrics & portfolio view

Backlog, resolution rate, 90-day flow and day/week deltas — for one project or compared across many.
