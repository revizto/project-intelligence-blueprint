# Skills manifest

## Ships in this bundle

| Skill | Role |
|---|---|
| `project-intelligence-dashboard` | Install action — deploys the dashboard artifact against the installer's own discovered licence/project. Authored for this plugin. |
| `skill-aeco-innovation-revizto` | Revizto platform knowledge (3D/2D, issues, stamps, clash, publishing, licensing). Curated copy — content sourced from Revizto Help Centre / Academy / developer.revizto.com. |
| `skill-aeco-innovation-revizto-api` | Revizto REST API v5, pyRevizto, MCP server, integration patterns. Curated copy — same public sources. Requires Revizto+. |

Cross-references inside the curated skills may name sibling skills (e.g. `revizto-issues`, `revizto-clash`, `integration-patterns`) that are **not** bundled — they are pointers to the wider skill library, not runtime dependencies. Nothing breaks in their absence.
